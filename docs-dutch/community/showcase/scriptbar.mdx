[ScriptBar](https://GitHub.com/raguay/ScriptBarApp) is een programma om de output van scripts of [Node-Red](https://nodered.org) server weer te geven. Het voert scripts uit die gedefinieerd zijn in het programma EmailIt en toont de output ervan. Scripts van xBar of TextBar kunnen worden gebruikt, maar momenteel werken alleen de TextBar-scripts goed. Het toont ook de output van scripts op uw systeem. ScriptBar plaatst ze niet in de menubalk, maar heeft ze allemaal in een handig venster voor eenvoudig bekijken. U kunt meerdere tabbladen hebben om verschillende dingen te tonen. U kunt ook de links naar uw meest bezochte websites behouden.
