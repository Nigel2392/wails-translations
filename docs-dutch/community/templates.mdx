# Sjablonen

Deze pagina dient als een lijst voor door de community ondersteunde sjablonen. Dien alstublieft een PR in (klik op `Bewerk deze pagina` onderaan) om uw sjablonen op te nemen. Om uw eigen sjabloon te bouwen, zie de [Sjablonen](../guides/templates.mdx) handleiding.

Om deze sjablonen te gebruiken, voer `wails init -n "Uw Project Naam" -t [de onderstaande link[@versie]]` uit.

Als er geen versieachtervoegsel is, wordt het sjabloon van de hoofdtakcode standaard gebruikt. Als er een versieachtervoegsel is, wordt het sjabloon van de code gebruikt die overeenkomt met de tag van deze versie.

Voorbeeld: `wails init -n "Uw Project Naam" -t https://github.com/misitebao/wails-template-vue`

:::warning Let op

**Het Wails-project onderhoudt, is niet verantwoordelijk en niet aansprakelijk voor sjablonen van derden!**

Als u niet zeker bent over een sjabloon, inspecteer dan `package.json` en `wails.json` om te zien welke scripts worden uitgevoerd en welke pakketten zijn geïnstalleerd.

:::

## Vue

- [wails-template-vue](https://github.com/misitebao/wails-template-vue) - Wails-sjabloon gebaseerd op de Vue-ecologie (geïntegreerde TypeScript, donker thema, internationalisatie, enkelvoudige pagina-routering, TailwindCSS)
- [wails-vite-vue-ts](https://github.com/codydbentley/wails-vite-vue-ts) - Vue 3 TypeScript met Vite (en instructies om functies toe te voegen)
- [wails-vite-vue-the-works](https://github.com/codydbentley/wails-vite-vue-the-works) - Vue 3 TypeScript met Vite, Vuex, Vue Router, Sass en ESLint + Prettier
- [wails-template-quasar-js](https://github.com/sgosiaco/wails-template-quasar-js) - Een sjabloon met JavaScript + Quasar V2 (Vue 3, Vite, Sass, Pinia, ESLint, Prettier)
- [wails-template-quasar-ts](https://github.com/sgosiaco/wails-template-quasar-ts) - Een sjabloon met TypeScript + Quasar V2 (Vue 3, Vite, Sass, Pinia, ESLint, Prettier, Composition API met &lt;script setup&gt;)
- [wails-template-naive](https://github.com/tk103331/wails-template-naive) - Wails-sjabloon gebaseerd op Naive UI (Een Vue 3 Componentenbibliotheek)
