# Routering

Routering is een populaire manier om weergaven te switchen in een applicatie. Deze pagina biedt enige begeleiding over hoe dat te doen.

## Vue

De aanbevolen aanpak voor routering in Vue is [Hash-modus](https://next.router.vuejs.org/guide/essentials/history-mode.html#hash-mode):

```js
import { createRouter, createWebHashHistory } from "vue-router";

const router = createRouter({
  history: createWebHashHistory(),
  routes: [
    //...
  ],
});
```

## Angular

De aanbevolen aanpak voor routering in Angular is [HashLocationStrategy](https://codecraft.tv/courses/angular/routing/routing-strategies#_hashlocationstrategy):

```ts
RouterModule.forRoot(routes, { useHash: true });
```

## React

De aanbevolen aanpak voor routering in React is [HashRouter](https://reactrouter.com/en/main/router-components/hash-router):

```jsx
import { HashRouter } from "react-router-dom";

ReactDOM.render(
  <HashRouter basename={"/"}>
    {/* De rest van je app komt hier */}
    <Routes>
      <Route path="/" element={<Page0 />} exact />
      <Route path="/page1" element={<Page1 />} />
      <Route path="/page2" element={<Page2 />} />
      {/* meer... */}
    </Routes>
  </HashRouter>,
  root
);
```
