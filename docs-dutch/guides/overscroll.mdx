# Overscroll

[Overscroll](https://developer.mozilla.org/en-US/docs/Web/CSS/overscroll-behavior) is het "bounce effect" dat je soms krijgt wanneer je voorbij de grenzen van de inhoud van een pagina scrolt. Dit is gebruikelijk in mobiele apps. Dit kan worden uitgeschakeld met behulp van CSS:

```css
html {
  height: 100%;
  overflow: hidden;
}
```
