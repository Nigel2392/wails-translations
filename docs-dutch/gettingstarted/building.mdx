# Uw project compileren

Voer vanuit de projectdirectory `wails build` uit.
Dit zal uw project compileren en de productieklare binair opslaan in de map `build/bin`.

Als u de binair uitvoert, zou u de standaardtoepassing moeten zien:

```mdx-code-block
<div class="text--center">
  <img
    src={require("@site/static/img/defaultproject.webp").default}
    width="50%"
    class="screenshot"
  />
</div>
<br />
```

Voor meer informatie over compilatieopties kunt u verwijzen naar de [CLI-referentie](../reference/cli.mdx#build).
