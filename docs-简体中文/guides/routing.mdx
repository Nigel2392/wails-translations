# 路由

路由是应用程序中切换视图的常见方式。本页提供了一些关于如何做到这一点的指导。

## Vue

Vue中路由的推荐方法是[Hash模式](https://next.router.vuejs.org/guide/essentials/history-mode.html#hash-mode)：

```js
import { createRouter, createWebHashHistory } from "vue-router";

const router = createRouter({
  history: createWebHashHistory(),
  routes: [
    //...
  ],
});
```

## Angular

Angular中路由的推荐方法是[HashLocationStrategy](https://codecraft.tv/courses/angular/routing/routing-strategies#_hashlocationstrategy)：

```ts
RouterModule.forRoot(routes, { useHash: true });
```

## React

React中路由的推荐方法是[HashRouter](https://reactrouter.com/en/main/router-components/hash-router)：

```jsx
import { HashRouter } from "react-router-dom";

ReactDOM.render(
  <HashRouter basename={"/"}>
    {/* 其余的应用程序放在这里 */}
    <Routes>
      <Route path="/" element={<Page0 />} exact />
      <Route path="/page1" element={<Page1 />} />
      <Route path="/page2" element={<Page2 />} />
      {/* 更多... */}
    </Routes>
  </HashRouter>,
  root
);
```
