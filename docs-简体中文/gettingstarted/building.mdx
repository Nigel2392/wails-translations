# 编译项目

从项目目录中运行`wails build`命令。
这将编译您的项目并将生产就绪的二进制文件保存在`build/bin`目录中。

如果您运行该二进制文件，您将看到默认的应用程序：

```mdx-code-block
<br />
```

有关编译选项的更多详细信息，请参阅[CLI 参考](../reference/cli.mdx#build)。
