# 开发你的应用程序

你可以通过在项目目录中运行 `wails dev` 来以开发模式运行你的应用程序。这将执行以下操作：

- 构建你的应用程序并运行它
- 将你的 Go 代码与前端绑定，以便可以从 JavaScript 中调用
- 利用 [Vite](https://vitejs.dev/) 的功能，在 Go 文件发生修改时进行重新构建/重新运行
- 设置一个 [web 服务器](http://localhost:34115)，通过浏览器提供应用程序。这使你可以使用你喜欢的浏览器扩展。你甚至可以从控制台调用你的 Go 代码

要开始，请在项目目录中运行 `wails dev`。关于此的更多信息可以在[这里](../reference/cli.mdx#dev)找到。
