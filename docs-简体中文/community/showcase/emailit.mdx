[EmailIt](https://github.com/raguay/EmailIt/) 是一个基于 Wails 2 的程序，它是一个基于 Markdown 的邮件发送程序，只有九个记事本、用于操作文本的脚本和模板。它还有一个脚本终端，可以在 EmailIt 上运行系统中的文件。这些脚本和模板可以直接从命令行使用，也可以与 Alfred、Keyboard Maestro、Dropzone 或 PopClip 扩展一起使用。它还支持从 GitHub 下载脚本和主题。文档尚未完整，但程序可以正常运行。它使用 Wails2 和 Svelte 构建，下载版本是一个通用的 macOS 应用程序。
