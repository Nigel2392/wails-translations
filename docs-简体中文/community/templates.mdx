# 模板

本页面作为社区支持的模板列表。请提交拉取请求（点击底部的“编辑此页面”）以包含您的模板。要构建自己的模板，请参阅[模板](../guides/templates.mdx)指南。

要使用这些模板，请运行`wails init -n "项目名称" -t [下面的链接[@版本]]`

如果没有版本后缀，默认使用主分支代码模板。如果有版本后缀，则使用与该版本标签相对应的代码模板。

例如：`wails init -n "项目名称" -t https://github.com/misitebao/wails-template-vue`

:::warning 注意

**Wails项目不维护、不负责、也不承担第三方模板的责任！**

如果对模板不确定，请检查`package.json`和`wails.json`，查看运行的脚本和安装的包。

:::

## Vue

- [wails-template-vue](https://github.com/misitebao/wails-template-vue) - 基于Vue生态的Wails模板（集成TypeScript、深色主题、国际化、单页面路由、TailwindCSS）
- [wails-vite-vue-ts](https://github.com/codydbentley/wails-vite-vue-ts) - 使用Vite的Vue 3 TypeScript模板（以及添加功能的说明）
- [wails-vite-vue-the-works](https://github.com/codydbentley/wails-vite-vue-the-works) - 使用Vite的Vue 3 TypeScript模板，包括Vuex、Vue Router、Sass和ESLint + Prettier
- [wails-template-quasar-js](https://github.com/sgosiaco/wails-template-quasar-js) - 使用JavaScript + Quasar V2的模板（Vue 3、Vite、Sass、Pinia、ESLint、Prettier）
- [wails-template-quasar-ts](https://github.com/sgosiaco/wails-template-quasar-ts) - 使用TypeScript + Quasar V2的模板（Vue 3、Vite、Sass、Pinia、ESLint、Prettier、Composition API with &lt;script setup&gt;）
- [wails-template-naive](https://github.com/tk103331/wails-template-naive) - 基于Naive UI的Wails模板（Vue 3组件库）
