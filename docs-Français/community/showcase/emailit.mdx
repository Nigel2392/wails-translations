[EmailIt](https://github.com/raguay/EmailIt/) est un programme Wails 2 qui est un expéditeur de courrier électronique basé sur Markdown avec seulement neuf blocs-notes, des scripts pour manipuler le texte et des modèles. Il dispose également d'un terminal de scripts pour exécuter des scripts dans EmailIt sur des fichiers de votre système. Les scripts et les modèles peuvent être utilisés à partir de la ligne de commande elle-même ou avec les extensions Alfred, Keyboard Maestro, Dropzone ou PopClip. Il prend également en charge les scripts et les thèmes téléchargés depuis GitHub. La documentation n'est pas complète, mais le programme fonctionne. Il est construit à l'aide de Wails2 et Svelte, et le téléchargement est une application universelle pour macOS.
