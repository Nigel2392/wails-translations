Le [lanceur Minecraft pour WarMine](https://warmine.ru/) est une application Wails qui vous permet de rejoindre facilement des serveurs de jeu modifiés et de gérer vos comptes de jeu.

Le lanceur télécharge les fichiers de jeu, vérifie leur intégrité et lance le jeu avec une large gamme d'options de personnalisation pour les arguments de lancement depuis le backend.

Le frontend est écrit en Svelte, le lanceur entier tient dans 9 Mo et prend en charge les versions de Windows 7 à 11.
