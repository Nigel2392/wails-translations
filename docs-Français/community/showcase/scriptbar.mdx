[ScriptBar](https://GitHub.com/raguay/ScriptBarApp) est un programme permettant d'afficher la sortie de scripts ou du serveur [Node-Red](https://nodered.org). Il exécute les scripts définis dans le programme EmailIt et affiche la sortie. Les scripts de xBar ou TextBar peuvent être utilisés, mais actuellement seuls les scripts de TextBar fonctionnent bien. Il affiche également la sortie des scripts sur votre système. ScriptBar ne les place pas dans la barre de menus, mais les affiche tous dans une fenêtre pratique pour une visualisation facile. Vous pouvez avoir plusieurs onglets pour afficher différentes choses. Vous pouvez également conserver les liens vers vos sites web les plus visités.
