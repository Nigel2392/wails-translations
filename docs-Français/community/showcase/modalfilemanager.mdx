[Modal File Manager](https://github.com/raguay/ModalFileManager) est un gestionnaire de fichiers à double volet utilisant les technologies web. Ma conception originale était basée sur NW.js et peut être trouvée [ici](https://github.com/raguay/ModalFileManager-NWjs). Cette version utilise le même code frontal basé sur Svelte (mais il a été grandement modifié depuis le départ de NW.js), mais le backend est une implémentation [Wails 2](https://wails.io/). En utilisant cette implémentation, je n'utilise plus les commandes `rm`, `cp`, etc. en ligne de commande, mais une installation de git doit être présente sur le système pour télécharger les thèmes et les extensions. Il est entièrement codé en Go et s'exécute beaucoup plus rapidement que les versions précédentes.

Ce gestionnaire de fichiers est conçu selon le même principe que Vim : des actions clavier contrôlées par l'état. Le nombre d'états n'est pas fixe, mais très programmable. Par conséquent, un nombre infini de configurations clavier peuvent être créées et utilisées. C'est la principale différence par rapport aux autres gestionnaires de fichiers. Il existe des thèmes et des extensions disponibles en téléchargement sur GitHub.
