# Compilation de votre projet

Depuis le répertoire du projet, exécutez `wails build`.
Cela compilera votre projet et enregistrera le binaire prêt pour la production dans le répertoire `build/bin`.

Si vous exécutez le binaire, vous devriez voir l'application par défaut :

```mdx-code-block
<div class="text--center">
  <img
    src={require("@site/static/img/defaultproject.webp").default}
    width="50%"
    class="screenshot"
  />
</div>
<br />
```

Pour plus de détails sur les options de compilation, veuillez vous référer à la [Référence de l'interface de ligne de commande (CLI)](../reference/cli.mdx#build).
