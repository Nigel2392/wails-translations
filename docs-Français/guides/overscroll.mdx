# Défilement excessif

Le "défilement excessif" est l'effet rebond que vous obtenez parfois lorsque vous faites défiler au-delà des limites du contenu d'une page. C'est courant dans les applications mobiles. Cela peut être désactivé en utilisant CSS :

```css
html {
  height: 100%;
  overflow: hidden;
}
```
