Boutons de souris

Le runtime Wails intercepte les clics de souris pour déterminer si une fenêtre sans cadre doit être redimensionnée ou si une fenêtre doit être déplacée.
Il a été demandé comment détecter quand un clic de souris s'est produit, car `window.onclick` ne signale pas correctement les boutons de la souris.
Le code suivant montre comment détecter les clics de souris:

```javascript
window.addEventListener("mousedown", handleMouseButtonDown);

function handleMouseButtonDown(event) {
  if (event.button === 0) {
    // bouton de souris gauche
  } else if (event.button === 1) {
    // bouton de souris du milieu
  } else if (event.button === 2) {
    // bouton de souris droit
  } else if (event.button === 3) {
    // bouton de souris arrière
  } else if (event.button === 4) {
    // bouton de souris avant
  } else {
    // autre bouton de souris
  }
}
```

Référence: https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent/button
